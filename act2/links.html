Elemento <a>

¿Para que se utiliza <a>?

La etiqueta HTML <a> define un hipervínculo.

La etiqueta tiene la siguiente sintaxis:

<a href="url"> texto_del_enlace </a>
El texto del enlace es la parte que será visible para el usuario.

Al hacer clic en el texto del enlace, se enviará al usuario a la dirección URL especificada.

Ejemplo

Un ejemplo utilizando el enlace de la página del TEC es el siguiente:
<a href="https://tijuana.tecnm.mx/""> Página del TEC. </a>
Este código da el siguiente resultado: Página del TEC.

El atributo href

El atributo más importante del elemento <a> es el atributo href, debido a que indica el destino del enlace.
Si el atributo href no está presente, la etiqueta no será un enlace.

El atributo target

Por default, la página vinculada se mostrará en la ventana actual del navegador. Para cambiar esto, debe especificar otro destino para el vínculo.
Es aquí donde entra el atributo target, el cual especifica donde abrir el documento vinculado. El atributo target puede tener uno de los siguientes valores:
  • _self. La opción por defecto, abre el documento en la misma ventana/pestaña en la que se hizo clic.
  • _blank. Abre el documento en una nueva ventana o pestaña.
  • _parent. Abre el documento en el marco principal.
  • _top. Abre el documento en el cuerpo completo de la ventana.
    •

Ejemplo

<a href="https://tijuana.tecnm.mx/" target="_blank""> Página del TEC. </a>

Imagenes como enlaces

Para poder utilizar imagenes como enlaces para acceder a otros sitios web, necesitamos colocar la etiqueta <img> dentro de la etiqueta <a>. Ejemplo:
<a href="https://tijuana.tecnm.mx/">
<img src="tecnm.png" alt="LogoTec">
</a>

Haga clic en en logo para ir a la página del TEC.

Referencias absolutas y relativas

Una referencia absoluta es aquella que tiene una URL completa de la página a la que se esta vinculando. Por el otro lado, una referencia relativa es un link local por lo que no utiliza toda la URL.

Ejemplo referencia absoluta

<a href="https://tijuana.tecnm.mx/""> Página del TEC. </a>

Ejemplo referencia relativa

<a href="imagenes.html"> Elemento <img>. </a>
Referencia absoluta (Lleva al sitio del TEC).

Referencia relativa (Lleva a la página del elemento img).

Titulos en enlaces

El atributo de title especifica información adicional sobre un elemento. La información generalmente se muestra como un texto de información sobre herramientas cuando el mouse se mueve sobre el elemento.

Ejemplo

<a href="https://www.google.com" title="Ir a Google""> Página de Google. </a>
Este código mostraria el siguiente enlace. Si se pasa el cursor encima, mostrará un texto:

Sitio de Google.
