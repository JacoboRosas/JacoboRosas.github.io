Elemento <p>

¿Qué es paragraph (párrafo)?

En HTML, un párrafo es un elemento que siempre comienza en una nueva línea y suele ser un bloque de texto. .

¿Cómo se definen los párrafos?

Los párrafos se definen con la etiqueta <p>.

Ejemplo de su uso

Si utilizamos el siguiente código:

<p> Primer parrafo. </p>
<p> Segundo parrafo. </p>

Nos dará el siguiente resultado:

Este es el primer texto de ejemplo.

Este es el segundo texto de ejemplo.

Visualización de párrafos en HTML

Al utilizar la etiqueta <p>, no podemos estar seguros de como se visualizara dentro de una página, ya que existen diferentes factores que podrían afectarle, como por ejemplo, pantallas grandes o pequeñas, resolución de la ventana, etc.
Sin embargo, si intentamos agregar espacios o líneas extras en nuestro código HTML el navegador automaticamente eliminará estos espacios y líneas extras cuando se visualicen en la página.

Un ejemplo de esto es al utilizar el siguiente código:

<p>
Este párrafo
contiene muchas líneas
en el código fuente,
pero el navegador
las ignora.
</p>

<p>
Este párrafo
contiene                   muchos espacios
en el código                 fuente,
pero el             navegador
las ignora.
</p>

Se mostrará el siguiente resultado, donde ignora por completo los espacios:

Este párrafo contiene muchas líneas en el código fuente, pero el navegador las ignora.

Este párrafo contiene muchos espacios en el código fuente, pero el navegador las ignora.

La etiqueta <hr>

Esta etiqueta define una ruptura temática en una página HTML y, con mayor frecuencia, se muestra como una línea horizontal. Su uso más común es para separar contenido dentro de una página.

La etiqueta <pre>

El texto dentro de un elemento <pre> conserva tanto los espacios como los saltos de línea.

Ejemplo de su uso

Si utilizamos el siguiente código:

<pre>

  Ejemplo de texto

  Esto es un ejemplo

  Este es otro ejemplo

  Último ejemplo

</pre>

Mostrará el siguiente resultado:

  Ejemplo de texto
  
  Esto es un ejemplo
  
  Este es otro ejemplo
  
  Último ejemplo
