¿Cómo agregar un lector-visor de documentos dentro de mi página?

Para poder agregar un lector-visor de documentos dentro de una página, es necesario hacer uso de la etiqueta <embed> , la cual define un contenedor para un recurso externo, como una página web, una imagen, un reproductor multimedia o una aplicación complementaria.
Para esta actividad, se utilizará para mostrar un archivo pdf.

Esta etiqueta tiene los siguientes parametros:

  • src - Especifica la ruta del archivo externo a embeber.
  • type - Especifica la categoría del contenido embebido.
  • width - Expecifica el ancho del contenido embebido.
  • height - Especifica el alto del contenido embebido.

Alternativamente, se puede utilizar la etiqueta <iframe> .

Utilizando la etiqueta <embed>

Una vez sabemos la función de la etiqueta, lo que sigue del procedimiento es sencillo, solo es necesario saber el nombre y dirección del archivo pdf que deseamos mostrar, en este caso, el nombre del archivo es LYA2.pdf y se encuentra en el directorio donde se esta la página, por lo que solo es necesario escribir su nombre en el atributo src de la etiqueta <embed> , indicar el tipo de contenido, y el ancho y alto que deseamos utilizar.

Para podar mostrar el visor PDF en está página, se utilizará el siguiente código:

<embed src="LYA2.pdf" type="application/pdf" width="800px" height="800px"/>

La etiqueta <iframe>

La otra alternativa es utilizar la etiqueta <iframe> , la cual también nos permitirá mostrar un archivo pdf dentro de nuestra página.
Para esto, se utilizará el siguiente código:

<iframe src="LYA2.pdf" type="application/pdf" width="800px" height="800px"/>

Esto mostraria lo mismo que la etiqueta <embed> , por lo que solo basta con observar el ejemplo anterior para ver el resultado.
